# Oppsett

1. Kjør `supabase/migrations/006_matches_and_predictions.sql` i Supabase SQL Editor.
2. Legg inn de nye repository-filene.
3. Erstatt `src/pages/tips/tips-page.js`.
4. Legg inn `src/styles/tips.css`.
5. Erstatt `src/styles/index.css`.
6. Deploy.
7. Logg inn, åpne `/tips`, lagre et tips og oppdater siden.

Tipsene skal fortsatt stå etter refresh.
